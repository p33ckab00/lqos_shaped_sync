# Advanced JSON + Field Guide UI Polish

v2.70.13-rc1 refines the admin/owner Advanced JSON workspace without changing config behavior.

## What changed

- Widens the Advanced JSON modal so the field guide has more room.
- Gives the field guide the larger side of the desktop split instead of compressing explanations.
- Makes the guide easier to scan with a wider path list, roomier detail pane, and aligned What / Why / When / Who / Where / How rows.
- Slightly reduces the JSON editor font size so more config fits on screen while keeping the guide readable.

## Safety

This is a UI polish release only. It does not change config save semantics, route permissions, field-guide meaning, generated files, scheduler timing, policy behavior, or LibreQoS apply mechanics.
