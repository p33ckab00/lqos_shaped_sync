# LQoSync Package Scoped Changes

This package keeps engine behavior as-is and only adds the requested operator-safety/documentation/UI ownership changes:

- Fresh install/update safety is clarified.
- Existing config.json, ShapedDevices.csv, and network.json are backup-first and preserve-existing by default.
- About is scoped to project identity and documentation links.
- Update Center owns version/update visibility and latest fetched commit summaries.
- Operator-facing repository/product language uses LQoSync.
- Compatibility service/path names remain unchanged where changing them would break existing installs.
