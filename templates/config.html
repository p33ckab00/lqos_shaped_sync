Selective MikroTik Collection

LQoSync bulk-reads MikroTik resources but requests only the fields needed for ShapedDevices.csv and network.json. This reduces RouterOS API payload size for larger client counts.

PPP PPPoE Read Order
  1. /ppp/active
  2. /ppp/secret
  3. /ppp/profile

Speed priority: secret comment → active comment → profile comment → profile name → profile rate-limit → default.

DHCP DHCP Read Order
  1. /ip/dhcp-server/lease
  2. /ip/dhcp-server name/comment
  3. DHCP server config

Speed priority: DHCP server comment → server name → server config speed → global default.

HS Hotspot Read Order
  1. /ip/hotspot/active
  2. /ip/hotspot/user
  3. /ip/hotspot/user/profile

Enhanced metadata can be disabled if fixed config speed is preferred.

About lqos_node_manager

lqos_node_manager is kept only for older LibreQoS installs that used it as a separate Web UI service. In newer LibreQoS setups, the important health checks are usually lqosd and lqos_scheduler. LQoSync will auto-hide lqos_node_manager when it is not installed, or mark it as legacy/optional when displayed.
