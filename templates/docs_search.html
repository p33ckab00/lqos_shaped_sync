Consolidated docs model
Source of truth: docs/content/*.md and docs/docs_manifest.json define the topic documentation used by WebUI search and GitHub docs.
GitHub view: README.md stays compact, FULL_DOCUMENTATION.md is the complete manual, and docs/DOCUMENTATION_INDEX.md is the topic map.
UI rule: Setup & Repair diagnoses problems and links here instead of duplicating long manuals. About stays lightweight.
Privacy: documentation search is local and read-only. It does not send queries outside the LQoSync WebUI.
