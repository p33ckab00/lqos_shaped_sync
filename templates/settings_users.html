Security notes

No database: users are saved in users.json, matching the LQoSync file-based design.

Passwords: never saved in plain text. New or changed passwords are saved as bcrypt hashes.

Safety: the UI prevents deleting yourself, deleting the last admin, or demoting the last admin.

Roles: admin can edit config/run actions. viewer can view dashboard, layouts, devices, logs, and status.
